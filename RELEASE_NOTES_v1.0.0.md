# Web Scraper Pro v1.0.0

First public release of the **Web Scraper Pro** Chrome extension.

## Description
Web Scraper Pro is a Chrome extension to quickly extract structured data from web pages,
using either guided CSS selector workflows or one-click quick scraping modes.

## Key Features
- Custom scraper with CSS selectors (container, title, link, price, description)
- **Quick Scrape** modes: links, titles, emails, phones, images, tables
- Smart selector auto-detection
- Preview before full extraction
- CSV export
- Copy results to clipboard
- Local scraping history with detailed JSON viewer
- Download settings (target folder and auto-download)
- Full English interface and documentation
- License: **GPL-3.0**
- Can support **OSINT** use cases when applied responsibly and lawfully on public sources

## Technical Notes
- Manifest V3
- Permissions: activeTab, scripting, storage, downloads
- Compatible with Chromium browsers (Chrome/Edge/Brave/Opera)
